## Description

Bierpass-Api tem como objetivo principal resolver o bar interno do Arromba e seus eventos, afim de trazer comodidade para quem quer comprar seu lanche ou bebida sem sair da sua mesa.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
